> **"Traditional AI learns how humans describe the world. Reflexive AI learns how its own decisions alter the world."**

Ouroboros is a comprehensive research program and cognitive architecture (V35) dedicated to **Reflexive Intelligence** — the capacity for AI systems to reason about their own causal impact on non-stationary environments.

---

## 📚 Research Program (The Six-Paper Framework)

We formalize reflexive intelligence through six interlinked papers. All source code and manuscripts are located in the [`ouroboros-papers/`](ouroboros-papers/) directory.

| ID | Paper Title | Target / Status | Key Contribution |
|:---|:---|:---|:---|
| **P1** | *Reflexive Intelligence* | Zenodo (Published) | SCRGN multi-tier reward formalization |
| **P2** | *Observer Depth (ReflexBench)* | **NeurIPS 2026 (Submitted)** | First benchmark for reflexive reasoning |
| **P3** | *When Rewards Collide* | **ICAIF 2026 (Backtesting)** | Multi-reward GRPO dynamics & RIP theory |
| **P4** | *Ouroboros V22 Architecture* | Zenodo (Published) | Recurrent Depth Cognitive Engine (RDCE) |
| **P5** | *The Cognitive Lifecycle* | Zenodo (Published) | 9-stage AI lifecycle & Memory Half-life |
| **P6** | *Cognitive Reward Topology* | Zenodo (Published) | 9-tier (SCRGNDWMT) reward architecture |

## 🧠 Technology Stack (V35)

- **Base Architecture**: Gemma-4-26B (Ouroboros Optimized)
- **Training Paradigm**: 12-Reward Group Relative Policy Optimization (GRPO)
- **Cognitive Engine**: Two-Pass "Decoupled Decision-First" Reasoning
- **Infrastructure**: Cloud-Local Hybrid (NVIDIA A800 + Mac 32GB Unified Memory)
- **Memory System**: 6-Layer Unified Memory with Hawkes-process temporal decay
